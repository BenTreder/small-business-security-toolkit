# Small Business Security Toolkit

A defensive cybersecurity toolkit designed for small businesses, freelancers, web developers, and website owners looking to improve website security and reduce common risks.

This repository focuses on practical website security improvements, Cloudflare protection, backup planning, WordPress hardening, and incident response basics.

---

## What This Repository Helps With

- Website hardening
- Login protection
- Rate limiting
- Cloudflare security
- Security headers
- WordPress security basics
- Backup planning
- Malware cleanup preparation
- Incident response planning
