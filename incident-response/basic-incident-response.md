# Basic Incident Response

What to do after a website compromise.

---

## Step 1

Take site offline if necessary.

---

## Step 2

Identify:

- Malware
- Unauthorized access
- File changes
- Admin changes

---

## Step 3

Restore clean backups if available.

---

## Step 4

Reset:

- Passwords
- Admin credentials
- API keys

---

## Step 5

Review logs.

---

## Step 6

Patch vulnerabilities.

---

## Step 7

Monitor for recurring issues.
