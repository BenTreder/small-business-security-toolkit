# Basic WordPress Security

Simple steps to improve WordPress security.

---

## Keep WordPress Updated

- Core updates
- Plugin updates
- Theme updates

---

## Use Strong Passwords

- Unique passwords
- Password managers
- MFA when possible

---

## Remove Unused Plugins

Delete:

- Unused plugins
- Old themes
- Inactive software

---

## Limit Login Attempts

Use:

- Rate limiting
- Cloudflare protections
- Login lockouts

---

## Backups

Always maintain:

- Daily backups
- Offsite backups
- Backup testing
