# Website Backup Checklist

Backups help reduce downtime during incidents.

---

## Back Up

- Website files
- Databases
- Media uploads
- Configuration files

---

## Backup Frequency

Small sites:

```txt
Weekly
```

Large sites:

```txt
Daily
```

Critical stores:

```txt
Multiple times per day
```

---

## Store Backups

- External storage
- Cloud storage
- Separate server locations

---

## Test Backups

Always verify backups can be restored.
