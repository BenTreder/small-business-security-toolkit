# Website Security Audit Checklist

## Basic Security

- HTTPS enabled
- SSL configured properly
- Security headers enabled
- Strong passwords used

---

## Website Security

- CMS updated
- Plugins updated
- Themes updated
- Unused plugins removed

---

## Access Security

- MFA enabled
- Login protections enabled
- Admin paths protected

---

## Backup Security

- Backups running
- Backups tested
- Offsite copies maintained

---

## Monitoring

- Logs reviewed
- Security alerts configured
- Malware scans performed
